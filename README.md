# Frontend Reservation Website

Website reservasi tempat olahraga (frontend only) yang dibangun sebagai bagian dari project tim.

## Role
Frontend Developer

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Halaman daftar venue olahraga
- Halaman booking jadwal
- Halaman pembayaran (UI)

## Notes
Project ini berfokus pada pengembangan tampilan (frontend) dan dijalankan secara lokal menggunakan XAMPP.
